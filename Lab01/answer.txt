- What happens if users choose “Cancel”?
=> If users choose "Cancel", it will return "You've chosen: No"
- How to customize the options to users, e.g. only two options: “Yes” and “No”, OR “I do” and “I don’t” (Suggestion: Use Javadocs or using Eclipse/Netbean IDE help).
=> To provide custom options (e.g., "Yes" and "No" only, or "I do" and "I don’t"), you can use JOptionPane.showOptionDialog().
